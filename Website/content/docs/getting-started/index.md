# Getting Started

OfficeIMO is a family of focused libraries rather than one giant abstraction. The fastest path is to choose the document format you need, install the matching package, and then move into the focused guide for that area.

## Pick Your Starting Point

| If you want to... | Start here |
|-------------------|------------|
| Install the right package or PowerShell module | [Installation](/docs/getting-started/installation/) |
| Build your first document in a few minutes | [Quick Start](/docs/getting-started/quickstart/) |
| Confirm framework, OS, or AOT support | [Platform Support](/docs/getting-started/platform-support/) |

## Choose A Product Area

- [Word Documents](/docs/word/) for `.docx` creation, editing, tables, images, bookmarks, and headers.
- [Excel Workbooks](/docs/excel/) for worksheets, tables, ranges, charts, and extraction.
- [PowerPoint](/docs/powerpoint/) for slide generation and presentation automation.
- [Markdown](/docs/markdown/) for fluent Markdown generation, parsing, and rendering.
- [CSV](/docs/csv/) for typed read/write workflows.
- [PSWriteOffice Docs](/docs/pswriteoffice/) for PowerShell automation on top of OfficeIMO.

## Recommended First Flow

1. Install the package you need from [Installation](/docs/getting-started/installation/).
2. Follow [Quick Start](/docs/getting-started/quickstart/) to generate a real file.
3. Move into the package-specific docs for the APIs you want to use regularly.
4. Use the [API Reference](/api/) when you need signatures, members, and source links.
