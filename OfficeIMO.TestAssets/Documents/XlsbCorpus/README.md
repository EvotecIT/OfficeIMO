# OfficeIMO XLSB Corpus

The files in `excel-generated` were created and saved by Microsoft Excel. They
exercise the BIFF12 package reader against independent, application-authored
workbooks rather than packages generated by OfficeIMO itself.

The fixtures cover values and shared strings, formulas, styles and dates,
worksheet geometry, merged cells, views, and internal and external hyperlinks.
Tests load them through the normal `ExcelDocument.Load(...)` surface, verify the
projected workbook model, and compare preserved binary package parts after
edits. OfficeIMO does not currently promise a complete native XLSB writer; the
tests distinguish editable projected content from preserved binary records and
parts.

Fixture identity and feature intent are recorded in
`../OfficeInteroperabilityCorpus/corpus-manifest.json`. Update the matching
SHA-256 only when a fixture is intentionally replaced, and update its focused
contract tests in the same change. Do not add customer documents or sensitive
data.
