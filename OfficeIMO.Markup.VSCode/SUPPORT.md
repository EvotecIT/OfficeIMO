# Support

Use the OfficeIMO GitHub issue tracker for support:

https://github.com/EvotecIT/OfficeIMO/issues

When reporting an OfficeIMO Markup for VS Code issue, include:

- the extension version
- your operating system and VS Code version
- the command you ran
- a minimal `.omd` or `.office.md` file that reproduces the issue
- any output shown by VS Code or the OfficeIMO Markup CLI

Security issues should be reported through the repository's security policy when available rather than a public issue.
